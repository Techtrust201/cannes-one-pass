# Guide utilisateur RX

1. Sélectionnez l’événement RX.
2. Identifiez l’exposant et son emplacement.
3. Vérifiez l’espace logistique proposé.
4. Choisissez les catégories concernées.
5. Choisissez la date de montage.
6. Choisissez un créneau disponible.
7. Ajoutez chaque véhicule et son gabarit.
8. Consultez les consignes affichées pour le véhicule.
9. Renseignez la reprise ou indiquez que le véhicule est identique.
10. Choisissez la date et le créneau de démontage.
11. Complétez les informations de manutention et confirmez les consignes.
12. Relisez puis envoyez la demande.

Un créneau complet est désactivé dans le formulaire. Une indisponibilité réseau n’empêche pas la saisie : le serveur contrôle systématiquement les capacités lors de l’envoi.

## Administration

Dans **Planning & quotas**, configurez d’abord les horaires, puis les capacités par zone et famille de véhicule. L’onglet **Processus véhicules** permet d’adapter les consignes léger/lourd affichées aux exposants.
